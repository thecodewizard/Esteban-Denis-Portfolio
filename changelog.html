Version 0 - The Alpha Stage

Version 0.1 - Beginning the Site

Version 0.1.1

In Version 0.1.1 we started this website. I decided to program this through the 'Mobile First' methodology, which begins developing on a phone and continues with phone landscape, tablet portrait, tablet landscape, computer and all above in that order.
The decision was made NOT to use any framework nor any backend. The purpose of this is purely advertisement.

Version 0.1.2

Version 0.1.2 Signifies the end of the mobile development stage as all the required content is present on the website. With all the content there, we can begin stage 2 where we expand the viewport from phone to landscape and then tablet.

Version 0.1.3

Version 0.1.3 signifies the start to end development phase of the phone landscape viewport. The site and all its content has been made in version 0.1.2 and as of now the only thing changing is the restructure of the html elements to fit inside the new viewports. We implemented the following functionality:

  • Tooltip on the skills to urge the user to 'click the circles to show the showcase'
  • Pull-to-scroll-back logic on the showcases

Version 0.2 - Expanding to Tablet

Version 0.2.1

Version 0.2.1 will signify the development for tablet on Portrait Mode. There is currently no additional features planned next to extending the viewport

Version 0.2.2

Version 0.2.1 will signify the development for tablet on Landscape Mode. Additional routemaps and functionality that is planned for this version are:

  • Host the site in Early Access - Alpha mode
  • Make temporary viewport for devices larger than tablet on Landscape and have it resizing the screen to tablet mode.

Version 0.3 - Expanding to other devices

Version 0.3.1

Version 0.3.1 will signify the development for laptop and desktop screens. Planned Functionality:

  • Delete the temporary Viewport from Version 0.2.2
  • Program support for screens larger than a laptop screen.
  • Program cross-browser support for Firefox and IE10+
  • Host the site in Bèta Mode - More Reachable than the Alpha Stage

Version 0.3.2

This site will be in Version 0.3.2 during peer review and spellcheck assessment mode. Meanwhile our developers will work on the following features:

  • Program cross-browser support for Opera and IE8+
  • Enable Lazy Loading for the Internship Images
